# レベルチェック 5級

英検5級の「単語」「短文語句空所」「会話文」「語句整序」の4項目について、公式の出題形式・レベル・話題をまとめ、本アプリで登録する問題の妥当性をチェックするためのドキュメントです。

**参照元**: 公益財団法人 日本英語検定協会 公式（試験内容・過去問）、旺文社「英語の友」等。  
**5級の目安**: 中学初級程度（中学1年レベル）、約600〜650語、英語学習を始めて最初の目標となる級。

---

## 1. 単語（語彙）

### 1.1 公式での位置づけ

- 英検5級の**リーディング筆記には「単語」のみの大問はない**。大問1は「短文の語句空所補充」15問で、そのうち**約7問が単語の知識を問う問題**、残りは文脈・文法とされる。
- **必要語数**: 約600〜650語程度（中学初級）。
- **出題傾向**: 名詞・動詞が多く、近年は前置詞・疑問詞も。曜日・月の名前など基礎的な単語の知識が求められる。
- **話題**: 家族、友達、学校、趣味、旅行、買い物、スポーツ、映画、音楽、食事、天気、道案内、自己紹介、休日の予定、近況報告など身近な事柄。

### 1.2 本アプリでの扱い

- **単語モジュール**（単語クイズ）で、級別に語彙を出題。5級は `vocabulary` テーブルの `level = '5級'` で管理。
- **シード**: `supabase/seed_vocabulary_5kyu.sql` に登録（約194語）。品詞・カテゴリ（基本動詞、時間、家族、場所・建物、食べ物・飲み物、スポーツ・楽器、様子・程度 など）付き。

### 1.3 チェック観点

- 語が**中学1年教科書レベル**か（drink, home, school, play, have など）。
- **品詞**: 名詞・動詞を中心に、形容詞・前置詞・疑問詞も含めて過不足がないか。
- 曜日（Sunday, Monday…）、月（January, August…）、数・時刻、家族・学校・身の回りの物など**5級でよく出るテーマ**をカバーしているか。
- 4級以上で習う語が混ざっていないか。

## 2. 短文の語句空所補充（大問1）

### 2.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **15問** |
| 形式 | 文脈に合う適切な**語句**を補う。短文または会話文。 |
| 問題文の種類 | 短文・会話文 |
| 解答形式 | **4肢選択**（選択肢は印刷） |

### 2.2 話題・レベル

- **場面**: 家庭、学校、地域（店舗・公共施設含む）、電話など。
- **話題**: 家族、友達、学校、趣味、旅行、買い物、スポーツ、映画、音楽、食事、天気、道案内、自己紹介、休日の予定、近況報告など。
- **文法・語法**: 現在形、命令文、現在進行形、can、基本語彙。中学1年程度。

### 2.3 本アプリでの扱い

- **リーディング → 短文の語句空所**（`/reading/short`）。`reading_short_questions` の `level = '5級'`, `question_type = 'short_fill'`。
- **シード**: `supabase/seed_reading_short_5kyu.sql`（短文のみ。会話形式の短いやりとりも含む）。1空所に `__BLANK__`、4肢選択、解説付き。

### 2.4 チェック観点

- 1文〜2文程度の**短い**文または短い会話であるか。
- 空所は**1語または1フレーズ**で、4肢から選ばせる形式か。
- 語彙・文法が**中学1年レベル**（現在形・進行形・can・基本動詞・曜日・天気・教科・家族など）に収まっているか。
- 話題が公式の「主な場面・題材」に沿っているか（家族、学校、趣味、スポーツ、食事、天気など）。

## 3. 会話文の文空所補充（大問2）

### 3.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **5問** |
| 形式 | **会話文**の空所に、適切な**文または語句**を補う。 |
| 問題文の種類 | 会話文 |
| 解答形式 | 4肢選択（選択肢は印刷） |

### 3.2 話題・レベル

- 本番は**短い2往復程度の会話**が中心。日常場面（挨拶、授業、買い物、道案内、時刻・日付のやりとり、依頼・承諾・断り、提案への返答など）。
- 中学初級の表現：Nice to meet you. / What time does the next train come? / At five o'clock. / All right. / Yes, let's. / Please open your book. など。

### 3.3 本アプリでの扱い

- **リーディング → 会話文の空所**（`/reading/conversation`）。`reading_short_questions` の `level = '5級'`, `question_type = 'conversation_fill'`。
- **シード**: `supabase/seed_reading_short_5kyu_conversation_50.sql`（50問）。話者表記は Teacher/Student, Woman/Man, Mother/Boy, Girl/Boy, Boy 1/Boy 2 など。1空所・4肢・解説付き。

### 3.4 チェック観点

- **会話文**（2人以上のやりとり）になっているか。
- 空所は**文または語句**で、会話の流れに合う応答・発話になっているか。
- 話者（A/B や Teacher/Student など）が明確で、5級らしい短い往復か。
- 挨拶・依頼・承諾・時刻・日付・提案（Let's ～ / Yes, let's.）など、**5級で頻出の会話表現**に沿っているか。

## 4. 日本文付き短文の語句整序（大問3）

### 4.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **5問** |
| 形式 | **日本文**を読み、その意味に合うように与えられた**語句を並べ替える**。 |
| 問題文の種類 | 短文 |
| 解答形式 | 並べ替え（本番では「1番目と3番目にくる語の番号を選ぶ」形式の回もある） |

### 4.2 出題傾向・文法

- **頻出パターン**: How much / How many 疑問文、What / Who / Where / When 疑問文、Does / Do 疑問文、肯定文（SVO・SVC、時・場所の前置詞句）、Let's ～ / Would you like ～、現在進行形（be + -ing）、助動詞 can。
- 語数は**3〜4語句**が中心。日本文と意味が一致する正しい語順にすることが目標。

### 4.3 本アプリでの扱い

- **リーディング → 語句整序**（`/reading/word-order`）。`reading_word_order_questions` の `level = '5級'`。
- **シード**: `supabase/seed_reading_word_order_5kyu_50.sql`（50問）。`prompt_ja`（日本文）、`words`（語句のJSON配列）、`correct_order`（正解の並びのインデックス）。解説は `seed_reading_word_order_explanations_5kyu.sql` で別UPDATE。

### 4.4 チェック観点

- **日本文**が明示され、その意味に合う**1文**が作れる並びになっているか。
- 語句の数は**3〜4語句**程度が5級らしいか（5語以上は負荷が高い）。
- 正解の英文が**文法的に正しく**、かつ日本文の意味と一致しているか（主語の有無、to不定詞の位置、Does/Do の後は動詞の原形など）。
- How much / How many、What/Who/Where/When、Does/Do、Let's、can、現在進行形など**5級の頻出パターン**をカバーしているか。

## 6. 登録問題チェックリスト（実務用）

新規登録・既存問題の見直し時に、以下を確認する。

### 単語

- [ ] 語が中学1年レベル（約600語程度の範囲）か
- [ ] 品詞・カテゴリが適切か
- [ ] 5級頻出テーマ（曜日・月・数・家族・学校・身の回り・スポーツ・食事など）を外していないか
- [ ] 4級以上の語が混在していないか

### 短文語句空所

- [ ] 15問形式に合わせた長さ・難易度か（短文または短い会話）
- [ ] 空所1つ・4肢選択か
- [ ] 話題が公式の場面・題材に沿っているか
- [ ] 文法が現在形・進行形・can・基本語彙の範囲か

### 会話文

- [ ] 会話文（2人以上のやりとり）になっているか
- [ ] 空所に当てはまる文・語句が会話の流れに合っているか
- [ ] 5問形式に合わせた長さ（短い2往復程度）か
- [ ] 5級頻出の会話表現（挨拶・時刻・日付・依頼・承諾・提案など）か

### 語句整序

- [ ] 日本文が明示され、意味に合う1文が作れるか
- [ ] 語句は3〜4語句程度か
- [ ] 正解の英文が文法的に正しく、日本文と一致しているか（主語・語順）
- [ ] 5級頻出パターン（How much/many、疑問詞、Does/Do、Let's、can、進行形など）を反映しているか
