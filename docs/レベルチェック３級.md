# レベルチェック 3級

英検3級の「単語」「短文語句空所」「会話文」「長文内容一致」の4項目について、公式の出題形式・レベル・話題をまとめ、本アプリで登録する問題の妥当性をチェックするためのドキュメントです。

**参照元**: 公益財団法人 日本英語検定協会 公式（試験内容・過去問）、英ナビ、4skills、英語塾ABC、Cheer up! English、キャタル、Kimini、eigonotomo 等。  
**3級の目安**: 中学卒業程度、約2,100語、5級・4級で習得した基礎力の集大成。リーディング・ライティング65分（リーディング25問＋ライティング2問）、リスニング約25分（30問）。3級から面接試験（スピーキング）が加わる。

---

## 1. 単語（語彙）

### 1.1 公式での位置づけ

- 英検3級のリーディングにも「単語」のみの大問はない。大問1「短文の語句空所補充」15問のうち、**約7問が単語の知識を問う問題**といわれる。語彙力と文脈判断が問われる。
- **必要語数**: 約**2,100語**程度（4級の約1,200語から約900語増）。単語力が合否を分ける要因の一つ。
- **出題傾向**: 名詞・動詞・形容詞が中心。中学2年〜3年で習うレベルの単語。例: invite（招待する）、stamp（切手）、delicious（おいしい）、question、subject、trip、ride、perform、choose、heavy、hungry、bright など。
- **話題**: 家族、友達、学校、趣味、旅行、買い物、スポーツ、映画、音楽、食事、天気、道案内、自己紹介、休日の予定、近況報告、海外の文化、**人物紹介、歴史**（3級で追加）。家庭・学校・地域（店舗・公共施設）・電話・アナウンスなどの場面。

### 1.2 頻出単語の例（3級レベル）

| カテゴリ | 例 |
|----------|-----|
| 動詞 | invite, ride, perform, choose, catch, learn, knock |
| 名詞 | question, subject, trip, experience, stamp, sound, fact |
| 形容詞 | delicious, heavy, hungry, bright, interesting |
| 熟語・表現 | How long have you ...?, take a vacation, knock on the door |

### 1.3 本アプリでの扱い

- **単語モジュール**で級別に語彙を出題。3級は `vocabulary` テーブルの `level = '3級'` で管理。
- 登録語は**中学卒業レベル（約2,100語範囲）**に収め、4級語と重複可。準2級以上の語が混在しないようにする。

### 1.4 チェック観点

- 語が**中学3年教科書レベル**か（invite, stamp, delicious, experience, perform など）。
- 品詞は名詞・動詞を中心に、形容詞・副詞・熟語も含めて過不足がないか。
- 3級でよく出るテーマ（人物紹介・歴史を除く4級と同様の場面＋人物紹介・歴史）をカバーしているか。
- 準2級以上で習う語が混ざっていないか。

## 2. 短文の語句空所補充（大問1）

### 2.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **15問** |
| 形式 | 文脈に合う適切な**語句**を補う。短文または会話文。 |
| 問題文の種類 | 短文・会話文 |
| 解答形式 | **4肢選択**（選択肢は印刷） |

### 2.2 話題・レベル

- **場面**: 家庭、学校、地域（各種店舗・公共施設を含む）、電話、アナウンスなど。
- **話題**: 家族、友達、学校、趣味、旅行、買い物、スポーツ、映画、音楽、食事、天気、道案内、自己紹介、休日の予定、近況報告、海外の文化、**人物紹介、歴史**など。
- **英文の長さ**: 1問あたりおおよそ**11〜25語**（17語程度の短文が多い）。
- **試される力**: 単語・熟語、会話表現、**文法（中学全範囲）**。語彙力が重要で、空所の前後にヒントがある。

### 2.3 本アプリでの扱い

- **リーディング → 短文の語句空所**（`/reading/short`）。`reading_short_questions` の `level = '3級'`、`question_type = 'short_fill'` または会話形式。
- 1空所に `__BLANK__`、4肢選択、解説付きで登録する想定。

### 2.4 チェック観点

- 1文〜2文程度の**短い**文または短い会話であるか（目安としておおよそ11〜25語）。
- 空所は**1語または1フレーズ**で、4肢から選ばせる形式か。
- 語彙・文法が**中学卒業レベル**（現在形・過去形・完了形・助動詞・比較・受け身・不定詞・動名詞・関係代名詞など中学全範囲）に収まっているか。
- 話題が公式の「主な場面・題材」に沿っているか。

## 3. 会話文の空所補充（大問2）

### 3.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **5問** |
| 形式 | **会話文**の空所に、適切な**文や語句**を補う。 |
| 問題文の種類 | 会話文 |
| 解答形式 | 4肢選択（選択肢は印刷） |

### 3.2 話題・レベル

- **形式**: 2人の会話の空欄部分に入る最も適切な言葉を4択から選ぶ。質問に対する応答、場面や会話の流れに合った発話、基本的な会話表現が試される。
- **会話のパターン**（4skills 等の解説より）:
  - **Q&Aパターン**: 疑問文（？）に続く空欄で、質問に答える形式。例：「How long have you had him?」→「Since I was ten.」
  - **説明パターン**: 肯定文の後に続く空欄で、詳しく説明する形式。例：「This sandwich is so delicious!」→「I just bought it from the store next to the station.」
- **解き方のコツ**: **消去法**。文脈に合わない選択肢を消していく。会話がつながるセリフを選ぶ。

### 3.3 本アプリでの扱い

- **リーディング → 会話文の空所**（`/reading/conversation`）。`reading_short_questions` の `level = '3級'`、`question_type = 'conversation_fill'`。
- 話者表記（Boy/Girl、Woman/Man など）を明確にし、1空所・4肢・解説付きで登録する想定。

### 3.4 チェック観点

- **会話文**（2人のやりとり）になっているか。
- 空所に当てはまる**文または語句**が、会話の流れ・質問への応答として自然か。
- 話者が明確で、3級らしい短い2〜3往復か。
- Q&Aパターン・説明パターンのいずれかに沿った**会話表現**（期間・理由・場所・提案・ねぎらいなど）か。

### 3.5 参考URL・例題（保存用）

| 種別 | URL | 内容 |
|------|-----|------|
| 公式 | https://www.eiken.or.jp/eiken/exam/grade_3/solutions.html | 会話文の空所補充 |
| 対策・解答のコツ | https://www.fourskills.jp/eiken3-conv | 4skills：会話の空欄補充で5点満点取るコツ・例題 |
| 対策講座 | https://www.ei-navi.jp/news/576/ | 英ナビ：3級 大問2 |

**保存した例題（4skills より）**

- Boy: Your dog is so cute! How long have you had him?  
  Girl: ( )  
  1 My dad gave it to me.  
  2 About five kilograms.  
  3 Long time no see.  
  4 **Since I was ten.**  
  正解：4（Q&Aパターン。「どれくらいの期間飼っているか」に対する答え）

- Woman1: This sandwich is so delicious! ( )  
  Woman2: I just bought it from the store next to the station.  
  1 How often do you go there?  
  2 Where is the station?  
  3 Where do we return it?  
  4 **Can you teach me how to make one?**  
  正解：4（A部分「駅の隣の店で買ってきた」に繋がる質問として「作り方を教えて」が自然）

- Woman: Your sister Misha seems very nice.  
  Man: Yes, she is. When we are together, ( )  
  1 she's never kind to me.  
  2 I'm three years younger than her.  
  3 **she always makes me happy.**  
  4 I have a brother too.  
  正解：3（説明パターン。「優しい姉/妹」の説明に合うのは3）

## 4. 長文の内容一致選択（大問3）

### 4.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **10問**（A掲示2問＋Bメール3問＋C説明文5問） |
| 形式 | パッセージの内容に関する質問に答える。 |
| 問題文の種類 | **掲示・案内**、**Eメール（手紙文）**、**説明文** |
| 解答形式 | 4肢選択（選択肢は印刷） |

### 4.2 各パートの詳細

| パート | 種類 | 長さ・形式 | 設問数 |
|--------|------|------------|--------|
| A | 掲示文（Notice） | タイトル含め12〜15行程度。Date, Time, Place などの小見出しに注意 | 2問 |
| B | メール文（E-mail） | 2〜3通のメール（2通目以降は返信）。From, To, Date, Subject を最初に確認 | 3問 |
| C | 説明文（Explanatory text） | 約25行前後。歴史上の出来事・人物、世界各地の行事がテーマ | 5問 |

### 4.3 話題・レベル

- **掲示・案内**: お知らせ、イベント案内、施設の利用案内など。
- **Eメール**: 友人・知人とのやりとり、勉強や活動の提案、返信で質問に答える形式など。質問文の続きを完成させる設問が含まれることもある。
- **説明文**: 事実関係、理由、手段、要旨を問う。中学卒業程度の語彙・文の長さ。
- **主な場面・題材**: 4級と同様＋**人物紹介、歴史**。

### 4.4 本アプリでの扱い

- **リーディング → 長文内容一致**（`/reading/long-content` 等）。長文パッセージと設問を `level = '3級'`、`passage_type = 'long_content'` で管理。
- 掲示・Eメール・説明文の3タイプをバランスよく登録する想定。

### 4.5 チェック観点

- 長文の**種類**が掲示・案内／Eメール／説明文のいずれかに沿っているか。
- 語彙・文の長さが**中学卒業レベル**か。
- 設問が「内容一致」型（事実・理由・手段・要旨・誰が何をしたか等）になっているか。
- Aは2問・Bは3問・Cは5問という**設問数の目安**に近い構成か（1パッセージあたりの設問数は問題集により可変）。
- 3級でよく出る話題（学校・趣味・体験・お知らせ・メール・人物紹介・歴史・世界各地の行事）に沿っているか。

| 傾向と対策 | https://english.cheerup.jp/article/7598 | Cheer up! English：3級リーディング3・掲示・メール・説明文 |

**保存した例題（長文・説明文のイメージ）**

- **掲示**: Date, Time, Place などがはっきりしたお知らせ形式。設問は「いつ」「どこで」「誰が対象か」など。
- **メール**: 2〜3通のやりとり。設問は「何についてのメールか」「返信で何を伝えたか」「質問の答えとして適切なのはどれか」など。
- **説明文**: 歴史上の出来事や人物、世界各地の行事。設問は事実・理由・要旨を選ばせる4択。

## 6. 登録問題チェックリスト（実務用）

新規登録・既存問題の見直し時に、以下を確認する。

### 単語

- [ ] 語が中学卒業レベル（約2,100語の範囲）か
- [ ] 品詞・カテゴリが適切か
- [ ] 3級頻出テーマ（人物紹介・歴史を含む場面・題材）を外していないか
- [ ] 準2級以上の語が混在していないか

### 短文語句空所

- [ ] 15問形式に合わせた長さ・難易度か（おおよそ11〜25語の短文または短い会話）
- [ ] 空所1つ・4肢選択か
- [ ] 話題が公式の場面・題材に沿っているか
- [ ] 文法が中学全範囲（過去・完了・比較・受け身・不定詞・動名詞・関係代名詞等）の範囲で適切か

### 会話文

- [ ] 会話文（2人のやりとり）になっているか
- [ ] 空所に当てはまる文・語句が会話の流れ（Q&Aまたは説明）に合っているか
- [ ] 5問形式に合わせた長さ（2〜3往復）か
- [ ] 消去法で解ける程度の**会話表現**（期間・理由・場所・提案・ねぎらい等）か

### 長文内容一致

- [ ] 掲示・案内／Eメール／説明文のいずれかの形式に沿っているか
- [ ] 語彙・文の長さが中学卒業レベルか
- [ ] 設問が内容一致型（事実・理由・手段・要旨等）か
- [ ] A2問・B3問・C5問の目安に近い構成か（アプリ上は1パッセージあたりの設問数は可変で可）
- [ ] 3級でよく出る話題（人物紹介・歴史・世界各地の行事を含む）に沿っているか
