# AiKen アプリコンセプト

## 名前の由来

**AiKen** は、次の3つを組み合わせた造語です。

| 要素 | 意味 |
|------|------|
| **AI** | 人工知能による学習サポート・添削・ヒント |
| **愛犬** | いつも側にいて寄り添ってくれる存在 |
| **英検** | 実用英語技能検定（対策の対象） |

---

## 目指すもの

英検対策が必要な生徒の皆さんに、**愛犬のようにいつも側にいて寄り添ってくれるアプリ**を目指しています。

- 大変な学習も「頑張ろう」と思えるように、**いつも味方でいる**アプリ
- 褒め・励まし・寄り添いを通じて、一人ひとりの学習を支える

このコンセプトに基づき、バディ（一緒に学習するキャラクター）の選定・コメント・ヒント表示などの仕様を設計しています。詳細は [BUDDY_DESIGN.md](./BUDDY_DESIGN.md) を参照してください。
