# 英検準1級 長文の語句空所補充 調査レポート

英検準1級の「長文の語句空所補充」（大問2）について、問題形式・レベル・出題傾向をまとめたレポートです。

**参照元**: 英検公式、過去問（添付画像6題）、e-LIFEWORK、eigonotomo 等

---

## 1. 問題形式（公式）

| 項目 | 内容 |
|------|------|
| 大問 | 大問2（長文の語句空所補充） |
| 問題数 | **6問**（2パッセージ × 3空所） |
| パッセージ数 | 2題 |
| 1パッセージの長さ | **250〜300語**程度 |
| 空所数 | 1パッセージあたり **3か所** |
| 選択肢 | 各空所 **4肢** |
| 問題文の種類 | 説明文・評論文など |
| 目標解答時間 | 約15分（1題あたり7〜8分） |

---

## 2. レベル・難易度

- **目安**: 大学中級程度
- **語彙**: 準1級レベル（約7,500〜9,000語程度）の範囲
- **文構造**: 複雑な従属節、関係詞、分詞構文、倒置など
- **特徴**: 空所補充問題は長文内容一致より語彙・表現が比較的易しめで、短時間で解くことが想定されている

---

## 3. 出題傾向

### 3.1 話題・題材

| カテゴリ | 例 |
|----------|-----|
| 歴史・社会 | ストライキ、密輸、剣闘士、エミュー戦争 |
| 科学・研究 | ワンヒットワンダー研究、音楽業界 |
| 教育・社会問題 | 低所得児童の早期教育 |
| 環境・自然 | 鳥類の衝突 |
| 文化・芸術 | ローマの剣闘士、エンターテインメント |

**傾向**: 社会生活一般、芸術、文化、歴史、教育、科学、自然・環境、医療、テクノロジー、ビジネス、政治など。日本国内だけでなく海外の話題も多い。

### 3.2 空所のタイプ（過去問分析）

| タイプ | 説明 | 例（選択肢） |
|--------|------|---------------|
| **接続表現** | 文頭のつなぎ言葉。論理関係（因果・対比・逆接・例示）を問う | In contrast, As a result, Consequently, Nevertheless, On the other hand, For instance, Therefore, In fact |
| **文脈・語彙** | 前後の文脈に合う語句・フレーズ | were able to avoid this, made little sense, remains a mystery, differed between schools |
| **推論・補完** | パッセージの流れから推論して補う | did not lessen the significance of, task would not be easy, audiences tend to reject them |
| **述部・動詞句** | 動詞部分や補語を補う | 名詞句、前の名詞を修飾する節を含む |

**重要**: ほぼ毎回、文頭の接続表現（つなぎ言葉）を問う問題が1問以上出題される。

### 3.3 パッセージの構造

- **タイトル**: 各パッセージにタイトルあり
- **段落**: 通常3〜4段落
- **論理**: 導入→展開→結論の流れが明確
- **空所の配置**: 各段落に1つずつ、または論理の転換点に配置

## 4. 過去問サンプル（添付画像より）

### 例1: One-Hit Wonders
- **Q22**: 研究者が何を調査したか → were able to avoid this
- **Q23**: 逆接の接続表現 → On the other hand
- **Q24**: 最初のヒット後の状況 → choice becomes crucial

### 例2: Striking Schoolchildren
- **Q19**: 因果の接続表現 → As a result
- **Q20**: 理由の性質 → was more complex
- **Q21**: 小規模参加と意義の関係 → did not lessen the significance of

### 例3: Bird Battle（エミュー戦争）
- **Q19**: 兵士の行動 → accepted the challenge
- **Q20**: 逆接の接続表現 → Consequently
- **Q21**: 軍事作戦の結果 → task would not be easy

### 例4: Gladiator Equality
- **Q22**: 論理の接続 → In fact
- **Q23**: 新規性の提供方法 → having women fight each other
- **Q24**: 上流階級女性の行動 → actively chose to become gladiators

### 例5: Smuggling in Cornwall
- **Q22**: 住民の状況 → faced economic hardship
- **Q23**: 政府の認識（however の前） → The problem was not new
- **Q24**: 追加の接続表現 → Moreover

### 例6: Early Education for Low-Income Children
- **Q19**: 逆接の接続表現 → Nevertheless
- **Q20**: プログラムの結果 → had the opposite effect
- **Q21**: 問題の核心 → how the children were taught

## 5. 本アプリでの扱い

- **テーブル**: `reading_passages`（passage_type = 'long_fill'）、`reading_passage_blanks`
- **空所表記**: 本文中に `__BLANK__` を配置
- **レベル**: `level = '準1級'`
- **1パッセージ**: 2〜3段落、約250〜300語、3空所

---

## 6. チェック観点

- [ ] パッセージが250〜300語程度か
- [ ] 空所が3か所か
- [ ] 各空所に4肢選択か
- [ ] 接続表現を問う問題が含まれているか
- [ ] 語彙・文構造が準1級レベルか
- [ ] 話題が公式の場面・題材に沿っているか
