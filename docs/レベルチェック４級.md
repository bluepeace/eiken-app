# レベルチェック 4級

英検4級の「単語」「短文語句空所」「会話文」「語句整序」「長文内容一致」の5項目について、公式の出題形式・レベル・話題をまとめ、本アプリで登録する問題の妥当性をチェックするためのドキュメントです。

**参照元**: 公益財団法人 日本英語検定協会 公式（試験内容・過去問）、ステューディアス、レアジョブ English Lab、ECC、Cheer up! English、studyrunups 等。  
**4級の目安**: 中学中級程度（中学2年レベル）、約800〜1,300語、5級の次に目指す級。リーディング35分（35問）・リスニング約30分（30問）。

---

## 1. 単語（語彙）

### 1.1 公式での位置づけ

- 英検4級のリーディングには「単語」のみの大問はない。大問1「短文の語句空所補充」15問のうち、**語彙力と文脈判断**が問われる。語彙問題と文脈・文法が混在する。
- **必要語数**: 約800〜1,300語程度（中学中級）。
- **出題傾向**: 名詞・動詞が中心。規則動詞（ask, call, enjoy, like, play, start）、不規則動詞（come, take, run, make, buy, cut, break, fly）など。学校・場所・職業・形容詞・副詞も頻出。
- **話題**: 家族、友達、学校、趣味、旅行、買い物、スポーツ、映画、音楽、食事、天気、道案内、自己紹介、休日の予定、近況報告、**海外の文化**など。電話・家庭・学校・地域（店舗・公共施設）・アナウンスなどの場面。

### 1.2 頻出単語の例（4級レベル）

| カテゴリ | 例 |
|----------|-----|
| 規則動詞 | ask, call, enjoy, like, play, start |
| 不規則動詞 | come, take, run, make, buy, cut, break, fly |
| 学校関連 | math, science, Japanese, dictionary, test, uniform, junior high school |
| 場所・建物 | library, hospital, restaurant, bank, airport |
| 職業 | teacher, doctor, nurse, farmer, pilot |
| 形容詞・副詞 | busy, famous, fast, often |

### 1.3 本アプリでの扱い

- **単語モジュール**で級別に語彙を出題。4級は `vocabulary` テーブルの `level = '4級'` で管理。
- 登録語は**中学2年レベル**に収め、5級語と重複しても可。3級以上の語が混在しないようにする。

### 1.4 チェック観点

- 語が**中学2年教科書レベル**か（member, story, contest, win, trip など）。
- 品詞は名詞・動詞を中心に、形容詞・副詞・前置詞も含めて過不足がないか。
- 4級でよく出るテーマ（学校・場所・職業・趣味・旅行・食事・天気など）をカバーしているか。
- 3級以上で習う語が混ざっていないか。

## 2. 短文の語句空所補充（大問1）

### 2.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **15問** |
| 形式 | 文脈に合う適切な**語句**を補う。短文または会話文。 |
| 問題文の種類 | 短文・会話文 |
| 解答形式 | **4肢選択**（選択肢は印刷） |

### 2.2 話題・レベル

- **場面**: 家庭、学校、地域（各種店舗・公共施設を含む）、電話、アナウンスなど。
- **話題**: 家族、友達、学校、趣味、旅行、買い物、スポーツ、映画、音楽、食事、天気、道案内、自己紹介、休日の予定、近況報告、海外の文化など。
- **文法・語法**: 現在形・過去形、進行形、助動詞（can, will 等）、基本語彙・熟語。中学2年程度。語彙力が最も重要で、ひっかけは少ない。

### 2.3 本アプリでの扱い

- **リーディング → 短文の語句空所**（`/reading/short`）。`reading_short_questions` の `level = '4級'`、適切な `question_type`。
- 1空所に `__BLANK__`、4肢選択、解説付きで登録する想定。

### 2.4 チェック観点

- 1文〜2文程度の**短い**文または短い会話であるか。
- 空所は**1語または1フレーズ**で、4肢から選ばせる形式か。
- 語彙・文法が**中学2年レベル**（過去形・助動詞・基本動詞・名詞・熟語）に収まっているか。
- 話題が公式の「主な場面・題材」に沿っているか。

## 3. 会話文の文空所補充（大問2）

### 3.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **5問** |
| 形式 | **会話文**の空所に、適切な**文や語句**を補う。 |
| 問題文の種類 | 会話文 |
| 解答形式 | 4肢選択（選択肢は印刷） |

### 3.2 話題・レベル

- **形式**: 2〜3ターンの短い会話の空所に入るセリフを4択で選ぶ。
- **求められる力**: 会話の流れを理解する力、日常表現の定型知識（挨拶・感謝・ねぎらい・申し出・承諾・断りなど）。
- **例**: 「お風呂掃除を終えたよ」→「よくやった」→「ゲームやっていいよ」のような、親子・友達のやりとり。「Thank you.」に対する「You're welcome.」、「何時に会う？」に対する「Around ten.」など。

### 3.3 本アプリでの扱い

- **リーディング → 会話文の空所**（`/reading/conversation`）。`reading_short_questions` の `level = '4級'`、会話形式。
- 話者表記（A/B、Mother/Boy など）を明確にし、1空所・4肢・解説付きで登録する想定。

### 3.4 チェック観点

- **会話文**（2人以上のやりとり）になっているか。
- 空所は**文または語句**で、会話の流れに合う応答・発話になっているか。
- 話者が明確で、4級らしい短い2〜3往復か。
- 挨拶・感謝・ねぎらい・依頼・承諾・時刻・提案など、**4級で頻出の会話表現**に沿っているか。

### 3.5 参考URL・例題（保存用）

| 種別 | URL | 内容 |
|------|-----|------|
| 公式 | https://www.eiken.or.jp/eiken/exam/grade_4/solutions.html | 会話文の文空所補充 |
| 対策テスト | https://syokyueigo.com/eiken4-2.html | 英検4級会話文対策テスト |
| 練習 | https://studyrunups.com/eiken_4_2_q/ | 4級 会話文補充 4択 |

**保存した例題（2024年度第3回過去問より・ステューディアス解説）**

- 息子が「お風呂掃除を終えたよ」と伝えた後、父親が「ゲームやっていいよ」と言う。その**前**に自然に入るセリフは？
  1 Take care.（気をつけて）❌ 別れのあいさつ向き  
  2 I don't know.（わからない）❌ 文脈に合わない  
  3 **Good job.**（よくやった）✅ ねぎらいとして自然  
  4 You, too.（あなたもね）❌ 感謝やあいさつへの返答用  

  正解：3

## 4. 日本文付き短文の語句整序（大問3）

### 4.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **5問** |
| 形式 | **日本文**を読み、その意味に合うように与えられた**語句を並べ替える**。 |
| 問題文の種類 | 短文 |
| 解答形式 | 並べ替えの結果、**「2番目と4番目にくる語の組み合わせ」を4択で選ぶ**形式がよく使われる |

### 4.2 出題傾向・文法

- **頻出**: 主語＋動詞＋補語・目的語の基本語順、時・場所・頻度は後ろに置く。疑問文では Do/Does、疑問詞（What, When, Where, How 等）の語順。熟語は「look at」「go to school」など1つの塊で覚えるとよい。
- **レベル**: 中学2年程度。比較的シンプルな文構造。
- **解き方のコツ**: 主語＋動詞を先に決める → 時間・場所・頻度を後ろに → よく出るセットを覚える → 消去法で「先頭に来ない」ものを外す。

### 4.3 本アプリでの扱い

- **リーディング → 語句整序**（`/reading/word-order`）。`reading_word_order_questions` の `level = '4級'`。
- `prompt_ja`（日本文）、`words`（語句のJSON配列）、`correct_order`（正解の並び）。解説は別テーブルまたは同一で管理。

### 4.4 チェック観点

- **日本文**が明示され、その意味に合う**1文**が作れる並びになっているか。
- 語句の数は**4〜5語句**程度が4級らしいか。
- 正解の英文が**文法的に正しく**、かつ日本文の意味と一致しているか（SVO・SVC、時・場所の順序など）。
- 過去形・助動詞・疑問文・熟語など**4級の頻出パターン**をカバーしているか。

### 4.5 参考URL・例題（保存用）

| 種別 | URL | 内容 |
|------|-----|------|
| 公式 | https://www.eiken.or.jp/eiken/exam/grade_4/solutions.html | 日本文付き短文の語句整序 |
| 解説 | https://studious.site/eiken4-question-pattern/ | 大問3 語順整序の解説・例題 |
| 攻略 | https://junpei-english.com/eiken-sentence-rearrangement-problems-grammar-word-order-beginners/ | 語句整序の文法・語順のコツ |
| 練習 | https://studyrunups.com/eiken_4_3_p/ | 4級 並べ替えプリント |

**保存した例題（2024年度第3回過去問より・ステューディアス解説）**

- 日本文（イメージ）:「数学のテストの間、教室はとても静かでした。」
- 語句: ① during the math test　② was　③ quiet　④ very　⑤ The classroom
- 正しい英文: **The classroom** (⑤) **was** (②) **very** (④) **quiet** (③) **during the math test.** (①)
- 2番目と4番目の語の組み合わせを選ぶ形式の場合の正解: 4（was と quiet など、実施回により出題形式は要確認）

## 5. 長文の内容一致選択（大問4）

### 5.1 公式の形式

| 項目 | 内容 |
|------|------|
| 問題数 | **10問**（長文は複数、各長文に2〜5問程度） |
| 形式 | パッセージの内容に関する質問に答える。 |
| 問題文の種類 | **掲示・案内**、**Eメール（手紙文）**、**説明文** |
| 解答形式 | 4肢選択（選択肢は印刷） |

### 5.2 話題・レベル

- **掲示・案内**: サッカークラブへの案内、お知らせなど。
- **Eメール**: 友人への勉強や活動の提案メールなど。
- **説明文**: 人物の趣味や経験、小学生〜高校生の生活・体験について（例：初めてのペット、学校行事など）。だれが・いつ・どこで・何をしたかを整理して読む。タイトルもヒントになる。
- **特徴**: ひっかけは少なく、読み取った内容に基づいて選択肢を選ぶ。時間配分の目安は約20分（1設問あたり約2分）。

### 5.3 本アプリでの扱い

- **リーディング → 長文内容一致**（`/reading/long` 等）。長文パッセージと設問を `level = '4級'` で管理。
- 掲示・Eメール・説明文の3タイプをバランスよく登録する想定。

### 5.4 チェック観点

- 長文の**種類**が掲示・案内／Eメール／説明文のいずれかに沿っているか。
- 語彙・文の長さが**中学2年レベル**か。
- 設問が「内容一致」型（本文の事実・理由・誰が何をしたか等）になっているか。
- 4級でよく出る話題（学校生活、趣味、体験、お知らせ・案内、メールのやりとり）に沿っているか。

### 5.5 参考URL・例題（保存用）

| 種別 | URL | 内容 |
|------|-----|------|
| 公式 | https://www.eiken.or.jp/eiken/exam/grade_4/solutions.html | 長文の内容一致選択 |
| 解説 | https://www.rarejob.com/englishlab/column/20210818_03/ | 4級レベル・長文の形式 |
| 練習（説明文） | https://english.cheerup.jp/article/7906 | 4級 長文読解2（説明文「Lucy's First Pet」） |

**保存した例題（説明文・Cheer up! English より）**

- **タイトル**: Lucy's First Pet  
- **概要**: Lucyは9歳。いとこのJane（10歳）の家でJaneや猫と遊ぶのが好き。ある日Janeの猫が4匹子猫を産んだ。Lucyは1匹もらえるかJaneの母に聞き、母から「親に聞いてね」と言われる。家に帰って母親に「毎日世話すると約束できる？」と聞かれ、「できる」と答えると許可され、大喜びする。

- (1) How old is Lucy? → 2) She is 9 years old.  
- (2) What does Lucy like to do? → 3) She likes playing with her cousin.  
- (3) What kind of animal will Lucy have? → 1) She will have a cat.  
- (4) Who will take care of Lucy's cat? → 3) Lucy will.  
- (5) Why did Lucy say "I'm the happiest girl in the whole world!"? → 3) Because she can have a pet.

## 7. 登録問題チェックリスト（実務用）

新規登録・既存問題の見直し時に、以下を確認する。

### 単語

- [ ] 語が中学2年レベル（約800〜1,300語の範囲）か
- [ ] 品詞・カテゴリが適切か
- [ ] 4級頻出テーマ（学校・場所・職業・趣味・旅行・食事・天気・海外の文化など）を外していないか
- [ ] 3級以上の語が混在していないか

### 短文語句空所

- [ ] 15問形式に合わせた長さ・難易度か（短文または短い会話）
- [ ] 空所1つ・4肢選択か
- [ ] 話題が公式の場面・題材に沿っているか
- [ ] 文法が現在形・過去形・進行形・助動詞・基本語彙の範囲か

### 会話文

- [ ] 会話文（2人以上のやりとり）になっているか
- [ ] 空所に当てはまる文・語句が会話の流れに合っているか
- [ ] 5問形式に合わせた長さ（2〜3往復）か
- [ ] 4級頻出の会話表現（挨拶・感謝・ねぎらい・依頼・承諾・時刻・提案など）か

### 語句整序

- [ ] 日本文が明示され、意味に合う1文が作れるか
- [ ] 語句は4〜5語句程度か
- [ ] 正解の英文が文法的に正しく、日本文と一致しているか
- [ ] 4級頻出パターン（SVO・時・場所・疑問文・熟語など）を反映しているか

### 長文内容一致

- [ ] 掲示・案内／Eメール／説明文のいずれかの形式に沿っているか
- [ ] 語彙・文の長さが中学2年レベルか
- [ ] 設問が内容一致型（事実・理由・誰が何をしたか等）か
- [ ] 4級でよく出る話題（学校・趣味・体験・お知らせ・メール）に沿っているか
