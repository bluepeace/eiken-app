# 英検4級 長文の内容一致選択 問題・解説集

本ドキュメントは `seed_reading_long_content_4kyu_50.sql` に登録された長文内容一致問題（約59問）の解説です。

## 英検4級 長文の内容一致のレベル・出題傾向

### 形式（本番試験）

- **大問4**: 長文の内容一致選択 … **全10問**
- **4A**: 掲示・案内 … 1題・**2問**（26–27）
- **4B**: Eメール（手紙文）… 1題・**3問**（28–30）
- **4C**: 説明文 … 1題・**5問**（31–35）

### レベル

- **対象**: 中学中級程度（中学2年程度）
- **語彙**: 筆記全体で約1,200語レベル。身近な単語・基本的な文法（現在・過去・未来、疑問詞、接続詞など）
- **テーマ**: 家庭・学校・地域・イベント・旅行・趣味など、身近な話題

### 出題傾向のポイント

| タイプ | 内容 | 問い方の傾向 |
|--------|------|----------------|
| **掲示・案内** | イベント、施設の案内、お知らせ | 日付・時刻・場所・持参物・期限などの**直接情報** |
| **Eメール** | 2通のやりとり（依頼・返事・計画） | **誰が・何を・いつ・どこで・なぜ**の具体情報、理由 |
| **説明文** | 短い物語・体験談（4–5段落） | 事実の**直接 recall**、理由・気持ち、**文完成**（〜because / 〜to ...） |

- 選択肢は本文の**言い換え**や**誤った数字・日付**が混ざるので、本文の該当箇所を押さえることが重要。
- 過去問では「読み込み中…」→ 級表示のチラつき対策として、本番同様に掲示2問・メール3問・説明文5問のバランスで作成しています。

## 4A 掲示・案内（Passage 1–8、各2問）

### Passage 1: School Library - Summer Hours

**問1** When does the library close in summer?  
**正解**: At 5:00 p.m.  
**解説**: 本文に "The library will be open from 9:00 a.m. to 5:00 p.m." とあるので、閉館は 5:00 p.m. です。

**問2** What should students do by August 25 if they want more books?  
**正解**: Return books.  
**解説**: "Please return books by August 25 if you want to borrow more." より、さらに借りるにはその日までに返す必要があります。

## まとめ

- **掲示**: 日付・時刻・場所・金額・持参物・期限を本文から正確に取り出す練習になります。
- **Eメール**: 誰の依頼／返事か、理由・条件（〜なので、〜の前に）を押さえることが大切です。
- **説明文**: 時系列（いつ・何をした・その結果）と、人物の希望・理由を押さえると正解しやすくなります。

本番の4級リーディングは35分で35問のため、長文部分では**先に設問を読んでから本文の該当箇所を探す**やり方も有効です。
