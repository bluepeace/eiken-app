# AiKen Web アプリ (eiken-app)

Next.js 14 + Supabase + OpenAI GPT-4o を用いた英検対策学習プラットフォームです。

**AiKen** は **AI**・**愛犬**・**英検**を組み合わせたコンセプトで、英検対策が必要な生徒に、愛犬のようにいつも側にいて寄り添い、大変な学習も頑張ろうと思える「いつも味方」のアプリを目指しています。詳細は [docs/CONCEPT.md](docs/CONCEPT.md) を参照してください。

## セットアップ

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

詳細は `QUICKSTART.md` と `PROJECT_DESIGN.md` を参照してください。
