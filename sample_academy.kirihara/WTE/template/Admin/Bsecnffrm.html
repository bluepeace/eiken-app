課金比率 %prof_caption%の配分 代理店の配分
通常完了 ％ ％ ％
%member_caption%通常キャンセル ％ ％ ％
%member_caption%緊急キャンセル ％ ％ ％
%member_caption%欠席キャンセル ％ ％ ％
  • 課金比率は、%member_caption%への課金額を算出するための比率になります。また、この比率から算出された金額は、%prof_caption%と代理店へ売上を配分する際の基となる金額にもなります。例えば課金比率を50%とすると、1000円のレッスンに対して500円が%member_caption%に課金され、この500円を所定の比率で%prof_caption%と代理店へ配分することになります。通常、通常完了の課金比率は100%とします。キャンセルの場合は、いわゆるキャンセル料を算出するための比率になります。
  • 課金比率により算出された課金額は、配分比率によって、%prof_caption%と代理店に配分されることになります。%prof_caption%と代理店の配分比率の和が100を超えることがないよう注意してください。
  • 上記の比率や配分の単位はすべてパーセントです。0～100の数値を指定してください。
  • 欠席キャンセルとは、%member_caption%がレッスンを予約したにもかかわらず受講せずにほったらかされた場合を指します。
レッスン
ヶ月先まで登録可能
  • ここで指定した月数だけ%prof_caption%は受入可能なスケジュールを登録することができ、また、%member_caption%も予約することができます。
  • 1～12までの月数を半角数字で指定してください。
  • 1を指定すると翌月末日まで登録可能です。
%member_caption%はレッスン開始より 前まで予約可能
  • 単位に日を指定した場合、レッスンの開始日時ではなく 00:00 が基準になります。1 日を指定した場合は、レッスン当日の前日の 00:00 になった時点で予約できなくなります。0 日を指定した場合は、レッスン当日の 00:00 になった時点で予約ができなくなります。
  • 単位に時間を指定した場合、レッスンの開始分ではなく 0 分が基準になります。1 時間を指定した場合は、レッスン開始時分の前の時間の 0 分になった時点で予約ができなくなります。例えば、レッスン開始時分が 13:30 なら 12:00 になった時点で予約ができなくなります。0 時間を指定した場合は、レッスン開始時の 0 分をもって予約ができなくなります。
  • レッスン開始時分直前まで予約を受け付けたい場合は、0 分を指定ください。
%member_caption%は 時間前まで通常キャンセル可能
  • 0～168までの時間を半角数字で指定してください。
  • 通常キャンセル可能な時間より直近にキャンセルした場合は、緊急キャンセル扱いとなり、%member_caption%から緊急キャンセル料に相当する割合を正規料金から徴収します。
レッスン終了時刻から 分以内
  • レッスンが終了してからここで指定した期間内に、%member_caption%と%prof_caption%はレッスン報告を行う必要があります。
  • この期間内にレッスン報告がない場合は、自動的にレッスンを受講したとして報告が記録され、それ以降は報告内容を変更することはできません。
  • 1～99999までの数値を半角で指定してください。
レッスン終了時刻から 分後に売上確定
  • レッスンが終了してからここで指定した期間を過ぎると売上が確定します。上記のレッスン報告有効期間より長い期間を指定してください。
  • %member_caption%と%prof_caption%のレッスン報告内容が不一致となった場合、または、いずれか一方が未報告だった場合、レッスン報告有効期間から、ここで指定する売上確定期間の間に、管理者はレッスン・ステータスを確定することになります。
  • もしレッスン・ステータスが未確定のまま、ここで指定する売上確定期間を過ぎると、自動的に所定のレッスン・ステータスがセットされ、売上が確定します。
  • 1～99999までの数値を半角で指定してください。

チケット最終購入日から 日後まで
  • 0～9999までの日数を指定してください。
  • 例えば、この有効日数が7日で、直近のチケット購入日が8月1日だったとすると、該当の%member_caption%の保持チケットは8月8日まで有効となり、8月9日には該当の%member_caption%の保持チケットは0円になります。
  • この日数は、スポット購入および自動課金のいずれにも適用されます。
  • 0を指定すると、自動的に最大値である9999日が適用されます。
最低 ヶ月
  • 1～12までの月数を指定してください。
  • 最低2ヶ月と設定すると、少なくとも、2ヶ月分の料金が課金されるまで自動課金を解約することができません。自動課金申込時に初月の課金が行われ、翌月の同日に2回目の課金が発生します。そのため、実質的に自動課金を申し込んだ日の翌月の同日を過ぎたら解約することができます。

  • 0 を指定するとリマインダーメールは送信されません。
